== Description ==

Discount for Enavto Customers with EDD Products plugin creates discount code for Easy Digital Download products for old customers from envato with their purchase code of a specific product.

== Installation ==

This section describes how to install the plugin and get it working.
 
e.g.
 
1. Upload `edd-discount-for-envato-customers` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Start using it by going to the Easy Digital Downloads -> Settings -> Extension menu from settings.
